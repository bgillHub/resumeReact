oneLiner

GitHub: github.com/bgillHub/oneLiner

Meteor-based project that emulates an anonymous message-board application that allows users to limit the responses given to their basic programming questions.

I felt that the responses you can find on Stack Overflow are often overwhelming or condescending, and there is no way for the user to express the level of expertise they would like in responding answers.

OneLiner allows users to specify a level of expertise for their question and anonymously post a question.

This application used to be hosted as a free Meteor web-app; if you would like to run it, please download it from the GitHub repo and run it in a Meteor environment.
